To run this application simply copy paste the extracted folder in your htdocs/www folder(depending on your server instalation).
First import the sql file provided to create the required database in your system.
Then goto the browser and browse to this folder on localhost.
To upload and store documents in the database, click upload tab and drag-drop your documents in the area provided.
To view the data stored in database, click the VIEW tab.

Created using HTML and Styled in CSS. Jquery handels all the ajax request to backend and handel basic animations.
PHP used to parse the data and store/fetch from database and generate HTML tags accordingly
